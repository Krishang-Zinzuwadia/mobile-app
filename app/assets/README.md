# Assets Directory

This directory should contain the following image assets for the Expo app:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image (2048x2048px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px or larger)

You can use any image editor or online tools to create these assets.
For now, the app will work without these assets but may show warnings.
